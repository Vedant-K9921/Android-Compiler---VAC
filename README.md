# Android-Compiler---VAC
Its a compiler application built for androids. Android-based Multi-Language Code Compiler Application. The app allows users to write, compile, and run programs directly on their mobile devices. It includes features like a code editor, syntax highlighting, output display, language selection, and file management.

System Analysis 
System analysis is the process of understanding the existing system and identifying the requirements of a new system. 
Existing System 
Currently, programmers rely on desktop-based IDEs such as: 
•	Android studio 
•	Visual Studio Code 
•	Atom 

These tools require installation and configuration, which may not be convenient for beginners. 
Problems in Existing System 
•	Requires powerful computer systems 
•	Installation and configuration complexity 
•	Limited portability 
•	Not suitable for quick coding practice on mobile devices 
Proposed System 

The proposed system is an Android-based Code Compiler Application that allows users to write and execute programs directly from their smartphones. 
Benefits include: 
•	Portable coding environment 
•	Easy to use interface 
•	Multiple language support 
•	Quick code testing 
 

Requirement Analysis 
Requirement analysis identifies the necessary hardware and software needed for the project. 
Hardware Requirements 
•	Android smartphone 
•	Minimum 4GB RAM 
•	Internet connectivity 
•	Processor: Quad-core or higher 
Software Requirements 
•	Android Studio 
•	Java Programming Language 
•	XML for UI design 
•	Online compiler API 
•	Android SDK
